## 질문 생성 도구 사용 가이드

질문을 생성하기 전에 도구를 사용해 맥락을 파악해.

### 워크플로우

1. `read_harvests`로 최근 수확물 확인 (5-10개)
2. `list_tags`로 전체 태그 분포 확인
3. 수집한 맥락을 바탕으로 9개 질문 생성

### 원칙

- **Crig 질문**: 최근 수확물에서 아직 명확히 정리되지 않은 개념을 골라 질문해.
- **Bisociate 질문**: 서로 다른 도메인 태그를 충돌시켜. `list_tags` 결과에서 먼 태그끼리 조합해.
- **Harvest 질문**: 최근 다루지 않은 영역에서 질문해. 태그 분포를 참고해 다양성을 확보해.
- 어제 수확물과 겹치는 질문은 피해.
